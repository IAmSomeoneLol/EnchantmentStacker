# Enchantment Stacker

Adds the ability to stack enchants such as Smite, Bane of arthopods, Sharpness (etc.) together. With ability to enable on armor aswell and on independent tools/weapons.

Giving the ability to control anvil costs, Enchantment levels, Ability to cross over /enchant limit beyond vanilla.

The Mod is still missing a few features, it will also support more version starting from 1.21.11 and upwards slowly.
The mod is purely server-side but can also be used within singleplayer worlds.

Grindstone can now Transfer enchants cleanly from whatever item was inputed inside the grindstone

(Fixes were made for Geyser support aswell)
